欢迎来到 LemonOS 宇宙终态全彩蛋缝合版！这是一个由单个 .html 文件构成的“操作系统”。它不需要安装、没有外部依赖，直接在你的浏览器里运行。无论是内核编译、黑客破解、3D 引擎还是挖矿打怪，它应有尽有。

🎮 内置现代软件与游戏
LemonEdge：内嵌式隔离浏览器，支持模拟上网。

Calculator / Notes / Paint：功能完好的计算器、记事本与画图板。

Lemon3D Engine：原生手写的伪 3D 光线追踪引擎，支持 W A S D 迷宫漫游。

Crystal Dash：内置 2D 挖矿与战斗小游戏。

Py Sandbox：内嵌 Pyodide WebAssembly 沙盒。
